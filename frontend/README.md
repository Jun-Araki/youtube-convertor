# YouTube Convertor Frontend (React)

A responsive React application for interacting with YouTube transcripts.

## Features

- **Interactive Transcript**: Side-bar captions that follow video playback.
- **Pinpoint Navigation**: Clicking any word in the transcript seeks the YouTube player to that exact millisecond.
- **Dynamic Highlighting**: CSS-driven highlighting of the current word being spoken.
- **Responsive Layout**: Designed to provide a comfortable reading and viewing experience side-by-side.

## Tech Details

- **Framework**: React 18+ with Vite.
- **Video Player**: `react-player` for YouTube integration.
- **Styling**: Vanilla CSS for maximum control over layouts and animations.

## Development

1. **Install Nodes**: `npm install`
2. **Environment**: Ensure the Rails backend is running on `http://localhost:3000`.
3. **Run Dev Server**: `npm run dev` (Runs on port 5173).
